There are four types of lists orderd, unorderd, nested, description lists.

Orderd list

  1. type can be used inside the initialization of lists so that the lists can be numbered in a specific way
  2. li attribute is used to insert list items
  3. ol is used to initialize orderd lists

Unorderd list

  • type can be used inside the initialization of lists so that the unoedered lists can be bulleted in a specific way
  • li attribute is used to insert list items
  • ul is used to initialize unorderd lists

Nested lists

We can insert unorderd list in unordered list aswell as in unorderd lists and vice versa, this is called nested lists
